## HealTrack - AI Chatbot 

The HealTrack chatbot is an interactive web-based virtual assistant built using Dialogflow ES and integrated directly into a custom website. It is designed to help users quickly access health-related information, answer common queries, and guide them through available features of the HealTrack platform.

The chatbot was styled and customized to match the branding theme of the website, including support for dark mode, smooth animation transitions, and UI elements that enhance user experience. It offers a modern interface based on the official Dialogflow Messenger widget, ensuring reliable and secure communication with the backend NLP engine.


## Key Features

- AI-Powered NLP using Google Dialogflow ES

- Dark Mode Support with animated theme transitions

- Smart Responses including greetings, FAQs, and small talk

- Custom UI Styling with branding, icons, and responsive layout

- Web Integration using the Dialogflow Messenger script

- Fully tested in local and hosted environments
